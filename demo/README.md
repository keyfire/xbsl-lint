# Demo project · Демо-проект

A tiny 1C:Element application to try the XBSL extension on: open this folder in VS Code
with the [XBSL extension](https://marketplace.visualstudio.com/items?itemName=keyfire.xbsl)
installed.

- `КарточкаЗадачи.xbsl` carries a handful of deliberate lint findings – watch the squiggles,
  hover them, apply the Quick Fix, configure a rule from the lightbulb.
- `КарточкаЗадачи.yaml` is a form: run *XBSL: form preview* (the button in the editor title),
  click the wireframe elements and edit their properties in the *Properties* panel.

---

Крошечное приложение 1С:Элемент для знакомства с расширением XBSL: откройте эту папку в
VS Code с установленным расширением.

- В `КарточкаЗадачи.xbsl` – несколько нарочных находок линтера: подчёркивания, hover,
  Quick Fix, настройка правила из лампочки.
- `КарточкаЗадачи.yaml` – форма: команда *XBSL: предпросмотр формы* (кнопка в заголовке
  редактора), клики по элементам каркаса и правка их свойств в панели "Свойства".
